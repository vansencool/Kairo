# Kairo

**Kairo** is a lightweight and fast 1.21+ mod for Hypixel SkyBlock.  
It brings useful Dungeons QoL features from 1.8 into 1.21.

---

## Current Features

- Blood Rush Reminder
- Blood Ready Notifier
- Leap Notifier
- Chat Filter (regex-based)
